# vim



- vi를 사용하는데 같은 키를 5번 이상 사용하면 그건 잘못하고 있는 것이다. 

- bash에서 vimtutor를 입력하면 사용법을 볼 수 있다. 



> ### w

한 단어 건너뛰기



> ### e

단어의 끝으로 이동



> ### b

단어의 앞으로 이동



> ### x

글자 지우기



> ### i

i를 눌르면 삽입모드로 변경되어 내용을 수정할 수 있다. 

esc를 누르면 일반 모드로 변경되고 일반모드에서 :를 입력하면 명령모드가 된다.



> ### w

저장



> ### q

빠져 나오기



> ### wq

저장 후 종료



> ### vi file

file 열기
